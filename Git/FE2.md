# 計算式の表し方
### 中置記法
- 数字の間に演算子を置く
- 例：１＋２

### 前置記法（ポーランド表記法）
- 数字の前に演算子を置く
- 例：＋１　２
- 数と数はスペースで区切る
- 計算の優先順位が高い演算子から順に前に出し、最後に括弧をとる

### 後置記法（逆ポーランド）
- 数字の後ろに演算子を置く
- 例：１　２＋
